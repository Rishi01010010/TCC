About TCC Detection System

Advanced AI/ML-based algorithm for Tropical Cloud Cluster identification and tracking

System Overview

The TCC Detection System is a comprehensive solution for identifying and tracking Tropical Cloud Clusters (TCCs) using half-hourly INSAT-3D satellite data over the Indian Ocean region. The system implements strict detection criteria, advanced independence algorithms, and time-based tracking capabilities.

Key Objectives
  • Automatic identification of convective cloud clusters
  • Extraction of 11 specific TCC features
  • Implementation of 1200km independence rule
  • Time-based tracking with adaptive search radii
  • Machine learning integration for pattern recognition
Technical Specifications
Data Requirements
  • Source: INSAT-3D IRBRT
  • Resolution: Half-hourly
  • Region: Indian Ocean (30°E-120°E, 40°S-30°N)
  • Format: NetCDF/HDF5
Detection Criteria
  • Temperature: IRBT ≤ 230K
  • Size: Radius ≥ 1° (~111km)
  • Area: ≥ 34,800 km²
  • Shape: Circular structure validation
Independence Algorithm
  • Separation: 1200km minimum distance
  • Grouping: Parent cluster identification
  • Validation: Distance matrix calculations
Extracted Features
Location Features
  1. Convective Latitude
  2. Convective Longitude
  3. Pixel Count
Temperature Features
  1. Minimum Temperature
  2. Mean Temperature
Geometric Features
  1. Maximum Radius
  2. Mean Radius
  3. Area (km²)
  4. Circularity
  5. Eccentricity
Physical Features
  1. Cloud Top Height

Tracking Algorithm

Time-based Search Radii

The tracking algorithm uses adaptive search radii based on time gaps between observations to account for cloud movement and evolution:

Time Gap Search Radius Purpose
3 hours 450 km Short-term tracking
6 hours 550 km Medium-term tracking
9 hours 600 km Extended tracking
12 hours 650 km Long-term tracking
